/user-server.js: Сервер для обработки сессий пользователей и их взаимодействий в многопользовательском режиме (например, логин, регистрация, управление данными пользователя).
/data-server.js: Сервер для хранения данных игры, таких как сохранения, достижения игрока и прогресс в игре.
/websocket.js: Логика WebSocket для обмена данными в реальном времени между сервером и клиентом (например, для многопользовательских функций).
