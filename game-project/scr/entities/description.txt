player.js: Логика для игрока: перемещение, атаки, взаимодействие с объектами, здоровье и другие механики.
monster.js: Логика для монстров: искусственный интеллект (AI), передвижение, атаки, цели и поведение.
npc.js: Логика для NPC (неигровых персонажей), включая диалоги, торговлю, квесты и другие взаимодействия.
