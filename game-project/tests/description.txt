/unit: Юнит-тесты для проверки функционала игры, например, правильность вычисления урона, обработки ввода и взаимодействий с объектами.
game.test.js: Тесты для проверки логики игры.
/integration: Интеграционные тесты для проверки работы разных систем игры вместе (например, генерация карт, переходы между уровнями).
map.test.js**: Проверка работы карт и уровней.
/ui: Тесты интерфейса и пользовательских взаимодействий.
menu.test.js: Проверка логики работы меню.
